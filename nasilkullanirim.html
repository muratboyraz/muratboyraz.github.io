    NASIL YAZARIM?


    CSS kodu yazarken nereye yazacağız diye soruyorsanız, cevabı 3 farklı yere yazabilirsiniz.Öncelikle her bir tagın içine ayrı ayrı yazabilirsiniz.Buna inline style yani satıriçi tarzı deniyor.Örneğin a tagını açıp içerisine CSS kuralı yazabilirsiniz.Fakat bu pek tercih edilen bir özellik değildir.Yani aslında sadece birkaç noktada bazı taglara CSS kuralı yazmak isterseniz makul fakat bütün noktalarda tek tek yazmak zahmetli ve gereksiz olacaktır.



    Diğer bir yöntem ise internal style yani içinde tarzı.Bu şekilde CSS kuralını yazarken ise üst tarafta head tagının içinde style tagı açılır ve arasına yazmak istediğimiz CSS kuralları yazılır.


    Son yöntemimiz ise external style yani dışında tarzı.Bu yöntemin özelliği yeni bir CSS dosyası oluşturup HTML sayfamıza onun adresini yazmaktır.Bu yöntemde oluşturulan örnek.css dosyasını HTML sayfamızda head tagı içerisine link href="örnek.css" rel="stylesheet" şeklinde yazarsak bütün HTML sayfamız örnek.css dosyasındaki özelliklerden etkilenir.Kodumuzun daha düzenli gözükmesi açısından tercih edilebilir bir yöntemdir.
